# dream-workflow

A working repository for collecting ideas, notes, markdown guides, and shell-based workflow experiments.

## Purpose

This project is a place to:

- capture thoughts worth revisiting
- document repeatable workflows in Markdown
- keep small `.sh` utilities that support those workflows
- evolve rough ideas into cleaner systems over time

## Structure

- `thoughts/` for raw notes and idea fragments
- `workflows/` for documented processes, checklists, and experiments
- `scripts/` for shell scripts that automate parts of the workflow

## Starting Point

The first version of this repo is intentionally simple. It is meant to grow as new workflows become worth saving, refining, and reusing.
